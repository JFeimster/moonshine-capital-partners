# FIE Public Data Policy

Anything inside this public partner-facing repo should be treated as public internet content.

## Public-safe data may include

- funding family labels
- borrower archetypes
- readiness dimensions
- generic review flag language
- white-labeled product/path summaries
- public-safe routing guidance
- public-safe recommendation weights
- embed and attribution documentation

## Do not publish here

- sensitive provider intelligence
- private affiliate economics
- full internal provider scoring weights
- source-confidence notes
- unresolved verification conflicts
- broker-only program notes
- full GPT orchestration prompts
- raw provider spreadsheets
- internal monetization priorities

Public route = lead capture and white-labeled guidance. Private FIE repo = lender intelligence moat.
