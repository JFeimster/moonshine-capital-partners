# Prequalification Analyzer Summary

The Business Loan Pre-Qualification Analyzer concept supports the FIE public route by separating intake-stage guidance from underwriting.

## It does

- review borrower-provided information
- identify likely strengths
- flag missing data
- identify review concerns
- recommend manual review when appropriate
- create structured internal summaries

## It does not

- approve loans
- replace underwriting
- pull credit
- verify bank statements
- validate tax returns
- confirm collateral
- guarantee funding options
