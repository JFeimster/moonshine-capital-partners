# Funding Readiness Scorecard Logic

The public FIE route uses the existing Funding Readiness Scorecard logic as the visible intake and scoring layer.

## Core score dimensions

- monthly revenue
- time in business
- profile strength
- bank activity
- business structure
- funding purpose

## Public tiers

| Score | Tier |
|---:|---|
| 80-100 | Highly Fundable |
| 65-79 | Fundable, But Needs Review |
| 45-64 | Possible Fit for Select Programs |
| 0-44 | Not Ready Yet - But Fixable |

Readiness is not approval. This tool does not verify bank statements, pull credit, validate revenue, or confirm provider criteria.
