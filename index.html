git init
git add .
git commit -m "Initialize Partner Ecosystem"
git branch -M main
git remote add origin YOUR_REPO_URL
git push -u origin main

To launch a new static partner URL (e.g. /partners/new-partner/index.html), duplicate our standard html structure and ensure you inject their corresponding reference tags into the header scripts.
